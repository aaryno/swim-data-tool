### Date Filtering
```python
# Season defined as Sept 1 (year-1) through Aug 31 (year)
# Example: Season 2025 = Sept 1, 2024 - Aug 31, 2025
def filter_by_season(df, season):
    df["swim_year"] = df["swim_date_parsed"].dt.year
    return df[df["swim_year"] == season]
```
